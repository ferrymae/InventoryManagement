
1.To run this project, you must have installed a virtual server XAMPP on your PC
2.Start Apache and MySQL in XAMPP
3.Extract file
4.Copy the main project folder
5.Paste in xampp/htdocs/
6.open a browser and go to URL “http://localhost/phpmyadmin/”
7.Click on the databases tab
8.Create a database naming “inventorymanagement” and then click on the import tab
9.Click on browse file and select “inventorymanagement.sql” file which is inside the “InventoryManagement” folder
10.Click on go.
After Creating Database
11.Open a browser and go to URL “http://localhost/InventoryManagement/”
